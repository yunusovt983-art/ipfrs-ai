# @cool-japan/ipfrs

WebAssembly bindings for IPFRS (Inter-Planet File RUST System).

## Install

```bash
npm install @cool-japan/ipfrs
```

## Usage

```javascript
const ipfrs = require('@cool-japan/ipfrs');
// Use WasmBlockStore, IpfrsNode, etc.
```

## Build from source

```bash
npm run build
```

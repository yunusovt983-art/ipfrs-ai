# IPFRS — Архитектура «Как это работает»

> **Назначение этой Wiki.** Объяснить, **как функционирует IPFRS** — от запроса
> пользователя до байтов на диске и обратно — строго в терминах
> **Domain-Driven Design (DDD)**. Каждое утверждение здесь выверено по реальному

## Что такое IPFRS за один абзац

**IPFRS (Inter-Planetary File Rust System)** — это контент-адресуемая
распределённая файловая система на Rust, расширенная слоем машинного интеллекта.
Любой кусок данных адресуется своим **CID** (`cid = hash(data)`), хранится как
неизменяемый **Block**, связывается с другими блоками в **Merkle-DAG**,
распространяется по P2P-сети (libp2p + Kademlia DHT), а сверх классического IPFS
добавлены два уникальных домена: **семантический поиск** (векторные эмбеддинги,
HNSW/DiskANN) и **нейро-символический вывод** (TensorLogic: 20+ движков
рассуждения + автоград + федеративное обучение).

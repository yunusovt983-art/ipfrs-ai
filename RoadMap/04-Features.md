## Функция #1: WebRTC сервер сигнализации ⭐⭐⭐⭐

Включить браузерные клиенты IPFRS через WebRTC.

### Задачи
1. REST API для обмена ICE кандидатами
2. Реле SDP offer/answer
3. Управление сеансом (WebSocket)
4. Обновить WASM привязки

### График
- Неделя 5-6: REST API сервера сигнализации
- Неделя 7: WASM привязки
- Неделя 8: Docker и документация

## Функция #2: S3-совместимый REST шлюз ⭐⭐⭐

Drop-in замена для S3 (объектное хранилище).

### API эндпойнты
```bash
PUT /bucket/object → ipfrs add
GET /bucket/object → ipfrs get
DELETE /bucket/object → ipfrs delete
HEAD /bucket/object → проверить наличие
LIST /bucket → список блоков
```

### Задачи
1. HTTP обработчики (Axum)
2. Аутентификация
3. Хранилище метаданных
4. Docker & документация
